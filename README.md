# DSiDesk

An app that allows you to view your Linux desktop on a Nintendo DSi.

## Overview

DSiDesk enables remote desktop viewing on your Nintendo DSi console, letting you access and interact with your Linux desktop from your portable handheld device.

## Features

- View your Linux desktop on Nintendo DSi
- Remote desktop streaming
- Cross-platform Linux support

## Requirements

- Nintendo DSi console
- Linux system
- Build tools (Make, GCC)

## Installation

### Building from Source

```bash
make
```

The project uses a Makefile for easy compilation. Ensure you have the necessary build tools installed:
- GCC or compatible C compiler
- GNU Make

### Setup

1. Clone the repository:
```bash
git clone https://github.com/MindlessLeader/DSiDesk.git
cd DSiDesk
```

2. Build the project:
```bash
make
```

3. Follow the specific setup instructions for your Nintendo DSi.
